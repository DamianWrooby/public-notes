## Semantyczne znaczniki HTML:
```html
<header>
Nagłówek bloku lub sekcji. Niekoniecznie nagłówek strony. Headerów może być kilka na jednej stronie.
Nie mogą być zagnieżdżone.
Nagłowek jakiejkolwiek treści.
</header>

<article>
Powtarzający się element jak np. blog posty w karuzeli.
</article>

<footer>Stopka. Podobnie jak header to stopka jakiejkolwiek odosobnionej treści, niekoniecznie całej strony.</footer>

<nav>Nawigacja</nav>

<main>Sekcja główna strony.</main>

<address>Służy do umieszczenia adresu.</address>

<section>Sekcja strony</section>

<aside>Sidebar</aside>

```
